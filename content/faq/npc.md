### Создать npc
```
.npc add [id]
```

### Узнать id
```
.lo cre [название]
```

Или с сайта https://ru.wowhead.com

### Сделать мирным
```
.mod faction 2007
```
Или
```
.npc set faction 2007
```

- 11 - Альянс
- 14 - Враждебные ко всем
- 85 - Орда
- 2007 - нейтральные
